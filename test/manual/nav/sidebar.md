# Sidebar (alternate branch)

[◂ Home](home.md)

Use this file to test that opening a **new** link discards forward history:

1. Home → Chapter One → Chapter Two (Forward history is now empty).
2. Press **Back** twice to return to Home (Forward now offers One/Two).
3. Click **[Sidebar](sidebar.md)** from Home — Forward should go **dim**,
   because this new navigation replaced the One/Two forward branch.

That mirrors how a web browser drops the forward stack when you take a new path.
